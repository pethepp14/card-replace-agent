# Card Replace Agent

A simple React + TypeScript starter for a card replacement request assistant.

## Features

- Request flow for broken, stolen, and damaged cards
- Simple support form and confirmation message
- Vite-based development workflow

## Run locally

1. npm install
2. npm run dev
3. Open the local Vite URL shown in the terminal

## Test data

Sample JSON input files are available in the data folder for testing the LangGraph flow:

- data/sample_card_request.json
- data/sample_card_request_broken.json
- data/sample_card_request_damaged.json

You can load these into your test script or API request body to simulate real card replacement cases.
